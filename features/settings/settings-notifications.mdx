## Подключение СМС-уведомлений

**1. Перейдите в раздел “Интеграции”** в личном кабинете.

![sms-1](https://guestme.ru/wp-content/uploads/2025/02/sms-1.png)

**2. Выберите СМС-шлюз**, с которым у вас заключен договор. Доступны:

- **SMS Aero**
- **SMSC**

**3. Введите данные для авторизации по API:**

- **SMS Aero**: Email, Ключ API

![sms-2](https://guestme.ru/wp-content/uploads/2025/02/sms-2.png)

- **SMSC**: Логин, Пароль

![sms-3](https://guestme.ru/wp-content/uploads/2025/02/sms-3.png)

Данные для интеграции можно узнать в личном кабинете соответствующего шлюза.

**4. Нажмите “Сохранить”**. Система проверит соединение.

- В случае ошибки появится сообщение с причиной.
- При успешном подключении – переход к настройке параметров.

## Настройка параметров работы SMS

#### Кому отправлять уведомления

- **Гостям, пришедшим через виджет** – уведомления отправляются только для броней, сделанных через виджет.
- **Всем гостям** – уведомления отправляются для всех броней ресторана.

![sms-4](https://guestme.ru/wp-content/uploads/2025/02/sms-4.png)

#### Выбор типов уведомлений

Можно включить или отключить отправку следующих сообщений:

- **Подтверждение номера при бронировании** – код подтверждения номера телефона (только для виджета).
- **Подтверждение бронирования** – после подтверждения резерва рестораном.
- **Напоминание о посещении** – за час до времени резерва.
- **Уведомление после визита** – после закрытия стола.
- **Отмена резерва** – при отмене бронирования.

![sms-5](https://guestme.ru/wp-content/uploads/2025/02/sms-5.png)

#### Настройка текстов уведомлений

Вы можете редактировать тексты сообщений, используя переменные для персонализации:

| **Переменная**           | **Значение**               |
| :----------------------- | :------------------------- |
| `<дата>`                 | Дата резерва (число.месяц) |
| `<время>`                | Время резерва (ЧЧ:ММ)      |
| `<сумма>`                | Сумма счета (рубли.коп)    |
| `<название ресторана>  ` | Название ресторана         |
| `<имя>`                  | Имя и фамилия гостя        |

**Тексты сообщений по умолчанию**

- **Подтверждение:**
  `<Название ресторана>. Ждем вас <дата> в <время>`
- **Напоминание:**
  `<Название ресторана>. Готовим ваш стол к <время>`
- **Уведомление после визита:**
  `<Название ресторана>. Спасибо за визит! Ваш счет составил <сумма> руб.`
- **Отмена:**
  `<Название ресторана>. Ваш резерв <дата> в <время> был отменен`

## Редактирование и удаление интеграции СМС

- **Редактировать интеграцию** можно, нажав на иконку карандаша, обновив данные и сохранив изменения.
- **Отключить или удалить интеграцию** можно через соответствующую кнопку с подтверждением во всплывающем окне.
- **Можно настроить несколько интеграций**, но активной будет только одна. Для смены активной интеграции нужно выбрать другую в списке и подтвердить действие.

![sms-6](https://guestme.ru/wp-content/uploads/2025/02/sms-6.png)

## Уведомления в мессенджеры через ChatPush

Теперь GuestMe поддерживает отправку уведомлений гостям с помощью **WhatsApp** и **Telegram**. Для этого используется сервис-партнер ChatPush.

Ниже описаны шаги, которые нужно выполнить на стороне ChatPush, чтобы получить данные для настройки интеграции в GuestMe.

<AccordionGroup>
  <Accordion title="Шаг 1. Зарегистрируйтесь в ChatPush">
    Перейдите на сайт сервиса и создайте аккаунт.
    <CardGroup>
      <Card title="Перейти на сайт ChatPush" href="https://chatpush.ru" />
    </CardGroup>
  </Accordion>
  <Accordion title="Шаг 2. Создайте канал">
    В личном кабинете ChatPush необходимо создать и настроить канал для того мессенджера, который вы планируете использовать.
    <Tabs>
      <Tab title="WhatsApp">
        1. Перейдите в раздел **"Каналы"** и нажмите **"Добавить канал"**.
        2. Выберите **WhatsApp** и следуйте инструкциям сервиса по подключению вашего бизнес-аккаунта WhatsApp Business API.
        3. После успешного создания канала он появится в вашем списке.
      </Tab>
      <Tab title="Telegram">
        1. Перейдите в раздел **"Каналы"** и нажмите **"Добавить канал"**.
        2. Выберите **Telegram**.
        3. Следуйте инструкциям сервиса. Как правило, это требует создания нового бота через официальный бот Telegram [@BotFather](https://t.me/botfather) и получения для него токена.
      </Tab>
    </Tabs>
  </Accordion>
  <Accordion title="Шаг 3. Скопируйте API ключ/токен канала">
    Это самый важный шаг. После создания канала зайдите в его настройки в личном кабинете ChatPush. Вам нужен **Токен API** (или API ключ).

    Скопируйте это значение — оно понадобится вам для настройки на стороне GuestMe.
    <Info>
    Главная задача на стороне ChatPush — получить **Ключ API** для нужного канала. Этот ключ используется для авторизации в GuestMe.
    </Info>
  </Accordion>
  <Accordion title="Шаг 4. Завершите настройку в GuestMe">
    Получив ключ API, вернитесь в личный кабинет GuestMe в раздел "Интеграции", выберите ChatPush и вставьте ваш Логин и скопированный ключ API в соответствующие поля.
    ![](/images/features/chatpush-integration-01.jpg "chatpush-integration")
    Далее вы можете настроить, кому и какие уведомления будут отправляться через ChatPush (по аналогии с настройкой СМС), и сохранить изменения.
  </Accordion>
</AccordionGroup>
