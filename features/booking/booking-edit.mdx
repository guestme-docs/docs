# Редактирование и отмена резерва

Для редактирования резерва нажмите на соответствующую строку в таблице.

![booking-table-edit](https://guestme.ru/wp-content/uploads/2020/10/booking-table-edit.jpg)

Вы можете отредактировать:

- **количество гостей**
- **выбрать другой стол, объединить столы и выбрать несколько**
- **изменить длительность резерва**
- **имя, телефон гостя (кроме резервов, созданных из виджетов ресторана)**
- **комментарий (кроме резервов, созданных из виджетов ресторана)**
- **длительность резерва**

![edit_booking](https://guestme.ru/wp-content/uploads/2020/10/edit_booking.png)

После завершения редактирования нажмите кнопку **“Сохранить изменения”**. Для отмены – **“Отменить резерв”**
